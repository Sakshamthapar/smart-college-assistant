# Smart College Assistant 🎓

A web-based assistant designed to help students manage their academic tasks efficiently.

## 🚀 Features
- Notes Management
- Timetable Planner
- AI Assistant (Simulated / Integrated)

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- Node.js (Backend)
- AI API (Gemini )
